
プログラムの設計
=============

### 関心の分離
シーンの描画のためのプログラムと、ゲームの処理を行うプログラムが混在するのはまずいので、
ファイルを分けて管理します。具体的には次のような分け方です。

~~~ sh
├── main.js  # プログラム実行のための窓口。内部で render/*.js と script/*.js を読む
├── render/  # 描画関連
│   ├── scene.js
│   ├── scene2.js
│   └── ...
└── script/  # プログラム関連
    ├── class.js
    ├── class2.js
    └── ...
~~~
